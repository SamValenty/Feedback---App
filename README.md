# Feedback---App
Este projeto consiste em uma aplicação web simples desenvolvida com **Node.js** e **Express.js** para coleta de feedbacks de alunos em um curso online.
<br>
O sistema permite cadastrar comentários, visualizar todos os feedbacks enviados e remover comentários da lista. Os dados são armazenados apenas em memória utilizando um array, sem uso de banco de dados ou APIs externas.
